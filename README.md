# choerodon-front-manager
Choerodon Front Manager是猪齿鱼微服务框架的服务管理中心前端，主要功能包括配置管理、swagger管理、API统计和菜单统计。

## Start

``` bash
cd choerodon-front-manager
npm install
chmod -R u+x node_modules
npm start
```
